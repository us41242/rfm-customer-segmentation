# Customer Segmentation Analysis (RFM)

## Executive Summary
This project analyzes customer behavior for a UK-based online retailer. Using RFM (Recency, Frequency, Monetary) analysis, we categorize customers to optimize marketing strategies.

## The Goal
* **Segment Customers:** Group users based on purchasing behavior.
* **Identify Trends:** Spot high-value cohorts and at-risk users.
* **Actionable Insights:** Drive retention strategies.

## The Data
Source: [UCI Machine Learning Repository - Online Retail II](https://archive.ics.uci.edu/dataset/502/online+retail+ii)
